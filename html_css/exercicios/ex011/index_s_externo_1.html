Comentário sobre o uso do Estilo Externo

Com a criação do arquivo em style.css faz com que todo o estilo que está sendo utilizado na página 1 e 2 fique nela. Porém sim podemos utilizar os style internos e inline no código HTML. E com isso o impacto daqueles estilos serão somente naquele código HTML que faz referência a aquela página

Sempre colocar o @charset "UTF-8" no inicio do arquivo .css

Para o arquvo ser utilizado ele é chamado no <head> do código html. link:css e basta colocar o nome do arquivo.css na href="".

        Ex.:
        <link rel="stylesheet" href="style.css">
