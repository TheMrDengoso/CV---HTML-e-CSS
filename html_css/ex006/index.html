Principais Formatações

Negrito / Destaque

Neste texto uma parte ficará em negrito utilizando a tag <strong>

Não utilizar mais o <b> para deixar o texto em negrito, pois pode ficar obsoleto e a tag <strong> deixa o texto com mais semântica

Itálico / Ênfase

Neste texto será uma parte que ficará em Itálico utilizando a tag <em>

Não utilizar mais o <i> para deixar itálico e sim a tag <em>, pois ela é mais semântica

Texto Marcado

Este texto iá ter alguma parte dele marcada que padrão é amarela. Porém o style no <head> está prioritário

Aqui irei trocar a cor da marcação pelo parâmetro style dentro da tag <mark> aqui é o exemplo.

Podemos também trocar a cor do marcado com o <style> no <head> aqui mesmo é um exemplo de cor trocada.

Texto Deletado

Aqui terá um parte do texto onde ficará uma parte em riscada como se tivesse sinalizado que foi feita uma correção sobre aquela parte do texto. Usamos o <del>

Texto Inserido

Neste trecho do código terá um parte onde estará sublinhado sinalizando que foi inserido ou que aquele trecho tenha certo destaque sobre os demais. Usamos a tag <ins>

Também existe a tag <u> porém não iremos mais utilizar.

Texto Sobrescrito

Aqui podemos utiliza para até caso utilizarmos fórmulas tipo X2-1 usando a tag <sup> deixa o texto Sobrescrito

Texto Subscrito

Aqui deixaremos tipo H20 ou CO2 com a tag <sub>
