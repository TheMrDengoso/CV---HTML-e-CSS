Principais Formatações

Negrito / Destaque

Neste texto uma parte ficará em negrito utilizando a tag <strong>

Não utilizar mais o <b> para deixar o texto em negrito, pois pode ficar obsoleto e a tag <strong> deixa o texto com mais semântica

Itálico / Ênfase

Neste texto será uma parte que ficará em Itálico utilizando a tag <em>

Não utilizar mais o <i> para deixar itálico e sim a tag <em>, pois ela é mais semântica
