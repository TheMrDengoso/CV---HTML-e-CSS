Comentário sobre o uso do Interno

A tag <style> fica na parte de cima do código no <head> e com isso colocamos quais são as tags que desejamos fazer os ajustes dentro da tag

Para fazer o ajuste na tag <body> precisamos chamar o body{} com chaves e dentro atribuir os estilos e os parâmetros para que sejão modificados, a mesma coisa vale para qualquer outro tipo de tag

        Ex.:
            <style>
                body{
                    background-color: lightskyblue;
                    font-family: Arial, Helvetica, sans-serif;
                    font-size: 20px;
                }
            </style>
