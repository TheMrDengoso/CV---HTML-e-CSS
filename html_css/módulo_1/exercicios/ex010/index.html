Imagens flexíveis

Abaixo está sendo utilizado a tag <picture> e dentro da tag a outra tag <source>

Dentro da tag <source> é onde está determinado em qual tamanho de pixels a imagem será transformada para que caiba melhor no navegador.

Audios em HTML 5

Iremos utilizar a tag <audio> para inserir audio no HTML 5

Nesta tag utilizamos os parametros de controls para que apareça os controles para o usuário manipular a música preload="metadata" para que seja carregado os controls e uma pequena parte inicial do audio. Mas o audio só vai ser carregado quando o usuário der play.

Vídeos em HTML 5

Iremos utilizar a tag de <video> para inserir video em HTML 5

Abaixo segue o video com de chá com o parâmetro width que serve para determinar o comprimento de largura do vídeo e também o parâmetro de controls para que o usuário consiga comandar o vídeo

Vídeos do Youtube no HTML 5

Com a ferramente de incorporar do Youtube conseguimos pegar o HTML que eles nos gera e inserir no código para que seja mostrado para o usuário.

Um ponto muito positivo é que não ocupa espaço e nem consome do servidor, pois está hospedado no Youtube.

A tag utilizada é a do <iframe> sempre é recomendado utilizar o próprio vídeo para que não tenha direitos autorais
